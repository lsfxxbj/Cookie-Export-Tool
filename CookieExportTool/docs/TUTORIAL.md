## 安装指南

### 安装步骤

1. 在 Chrome 浏览器中打开 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择包含此项目的文件夹
5. 扩展安装成功后，会在浏览器工具栏中显示图标

### 权限说明

安装时浏览器会提示扩展需要以下权限：

- `cookies`: 访问浏览器 cookies
- `activeTab`: 获取当前活动标签页信息
- `storage`: 保存用户设置（如语言偏好）
- `<all_urls>`: 获取所有网站的 cookies

这些权限仅用于实现扩展功能，不会收集或传输任何用户数据。

## 基本使用

### 打开扩展

点击浏览器工具栏中的扩展图标即可打开弹窗界面。

### 界面介绍

扩展界面分为两个标签页：

1. **导出标签页**：用于导出 Cookie
2. **导入标签页**：用于导入 Cookie

### 语言切换

在界面顶部可以选择语言：
- 中文
- English
- Français

## 导出 Cookie

### 导出当前网站 Cookie

1. 访问目标网站
2. 打开扩展弹窗
3. 在导出标签页中选择需要的导出格式
4. 点击"导出当前网站 Cookie"按钮
5. Cookie 将被下载为文件

### 导出所有 Cookie

1. 打开扩展弹窗
2. 在导出标签页中选择需要的导出格式
3. 点击"导出所有 Cookie"按钮
4. Cookie 将被下载为文件

### 复制 Cookie 到剪贴板

1. 打开扩展弹窗
2. 在导出标签页中选择需要的导出格式
3. 点击"复制当前网站 Cookie"或"复制所有 Cookie"按钮
4. Cookie 内容将被复制到剪贴板

### 导出格式

支持以下四种导出格式：

1. **JSON**：JavaScript 对象表示法，易于程序处理
2. **CSV**：逗号分隔值，适用于电子表格软件
3. **XML**：可扩展标记语言，结构化良好
4. **Netscape Cookie**：Netscape Cookie 格式，兼容 curl 等工具

## 导入 Cookie

### 通过文件选择导入

1. 打开扩展弹窗
2. 切换到导入标签页
3. 点击"浏览文件"按钮选择 Cookie 文件
4. 文件将自动导入

### 通过拖放导入

1. 打开扩展弹窗
2. 切换到导入标签页
3. 将 Cookie 文件拖放到指定区域
4. 文件将自动导入

### 支持的导入格式

支持导入以下四种格式的 Cookie 文件：

1. **JSON**
2. **CSV**
3. **XML**
4. **Netscape Cookie**

## 格式说明

### JSON 格式

```json
[
  {
    "name": "cookie_name",
    "value": "cookie_value",
    "domain": ".example.com",
    "path": "/",
    "expirationDate": 1234567890,
    "httpOnly": false,
    "secure": true,
    "session": false,
    "sameSite": "lax"
  }
]
```

### CSV 格式

```
domain,flag,path,secure,expiration,name,value
".example.com",FALSE,"/",TRUE,1234567890,"cookie_name","cookie_value"
```

### XML 格式

```xml
<?xml version="1.0" encoding="UTF-8"?>
<cookies>
  <cookie>
    <domain>.example.com</domain>
    <flag>FALSE</flag>
    <path>/</path>
    <secure>TRUE</secure>
    <expiration>1234567890</expiration>
    <name>cookie_name</name>
    <value>cookie_value</value>
  </cookie>
</cookies>
```

### Netscape Cookie 格式

```
# Netscape HTTP Cookie File
# http://curl.haxx.se/rfc/cookie_spec.html
# This file was generated by Cookie Export Tool

.example.com	FALSE	/	TRUE	1234567890	cookie_name	cookie_value
```

## 过滤功能

### 按 Cookie 类型过滤

在"Cookie 类型"下拉菜单中可以选择：

1. **所有**：导出所有 Cookie
2. **HTTPS**：仅导出安全 Cookie（secure）
3. **HTTP**：仅导出非安全 Cookie

## 常见问题解答

### 1. 为什么导出的 Cookie 文件为空？

可能原因：
- 当前网站没有 Cookie
- 过滤条件过于严格，没有匹配的 Cookie
- 网站设置了 HttpOnly 标志，无法通过 JavaScript 访问

解决方法：
- 确认网站确实有 Cookie
- 检查过滤条件
- 尝试访问其他网站

### 2. 为什么导入 Cookie 时提示格式错误？

可能原因：
- 文件格式不正确
- 文件内容损坏
- 文件编码问题

解决方法：
- 确认文件是从本工具导出的或符合指定格式
- 检查文件是否完整
- 确保文件使用 UTF-8 编码

### 3. 为什么复制 Cookie 到剪贴板失败？

可能原因：
- 浏览器安全策略限制
- 系统剪贴板权限问题

解决方法：
- 确保在用户交互（如点击按钮）后才尝试复制
- 检查浏览器是否允许网站访问剪贴板

### 4. 如何更新扩展？

开发模式下：
1. 替换项目文件
2. 在 `chrome://extensions/` 页面点击扩展的"重新加载"按钮

商店版本：
- Chrome 会自动更新扩展

### 5. 扩展会收集我的数据吗？

不会。此扩展不收集、存储或传输任何用户数据。所有操作都在本地完成。

### 6. 导出的 Cookie 安全吗？

导出的 Cookie 文件包含敏感信息，应该妥善保管，避免泄露给他人。

### 7. 如何卸载扩展？

1. 在 Chrome 浏览器中打开 `chrome://extensions/`
2. 找到 Cookie Exporter 扩展
3. 点击"移除"按钮

### 8. 支持哪些浏览器？

目前仅支持 Google Chrome 浏览器，需要 88 版本或更高版本。
