A powerful Chrome extension that can export browser cookies in multiple formats (JSON, CSV, XML, Netscape Cookie) and supports import functionality.

## Features

- Export all cookies from the current website
- Export all cookies from the browser
- Support multiple export formats: JSON, CSV, XML, Netscape Cookie
- Support downloading exported cookies as files
- Directly copy cookies to clipboard in the extension interface
- Support importing cookie files (JSON, CSV, XML, Netscape formats)
- Multilingual support: Chinese, English, French
- Support filtering by Cookie type (HTTP/HTTPS)
- Responsive design, adaptable to different screen sizes

## Installation

1. Open `chrome://extensions/` in Chrome browser
2. Enable "Developer mode" in the upper right corner
3. Click "Load unpacked"
4. Select the project folder

## Usage

1. Click the extension icon in the browser toolbar to open the popup
2. In the export tab:
   - Select export format (JSON/CSV/XML/Netscape Cookie)
   - Select Cookie type (All/HTTPS/HTTP)
   - Click "Export Current Site Cookies" to export cookies from the current website
   - Click "Export All Cookies" to export all cookies
   - Use "Copy Current Site Cookies" or "Copy All Cookies" buttons to copy content to clipboard
3. In the import tab:
   - Drag and drop or select Cookie file to import
   - Click "Import Cookies" button

## Output Formats

### JSON Format
```json
[
  {
    "name": "cookie_name",
    "value": "cookie_value",
    "domain": ".example.com",
    "path": "/",
    "expirationDate": 1234567890,
    "httpOnly": false,
    "secure": true,
    "session": false,
    "sameSite": "lax"
  }
]
```

### CSV Format
```
domain,flag,path,secure,expiration,name,value
".example.com",FALSE,"/",TRUE,1234567890,"cookie_name","cookie_value"
```

### XML Format
```xml
<?xml version="1.0" encoding="UTF-8"?>
<cookies>
  <cookie>
    <domain>.example.com</domain>
    <flag>FALSE</flag>
    <path>/</path>
    <secure>TRUE</secure>
    <expiration>1234567890</expiration>
    <name>cookie_name</name>
    <value>cookie_value</value>
  </cookie>
</cookies>
```

### Netscape Cookie Format
```
# Netscape HTTP Cookie File
# http://curl.haxx.se/rfc/cookie_spec.html
# This file was generated by Cookie Export Tool

.example.com	FALSE	/	TRUE	1234567890	cookie_name	cookie_value
```

## Permission Information

- `cookies`: Access browser cookies
- `activeTab`: Get information about the current active tab
- `storage`: Save user settings (such as language preferences)
- `<all_urls>`: Get cookies from all websites

## Notes

- This extension does not collect or transmit any user data
- Exported cookies contain sensitive information, please keep them safe
- Only use this tool on websites where you have the appropriate permissions
- The import function will add new Cookies to the browser, please ensure the source is trusted

## Modular Architecture

This project adopts a highly modular design with main modules including:

- **Core Initialization Module**: [popupInitializer.js](js/modules/popupInitializer.js)
- **Operation Modules**: 
  - [cookieOperations.js](js/modules/cookieOperations.js) - Cookie-related operations
  - [exportOperations.js](js/modules/exportOperations.js) - Export-related operations
  - [fileOperations.js](js/modules/fileOperations.js) - File-related operations
  - [importManager.js](js/modules/importManager.js) - Import-related operations
- **Functional Modules**:
  - [errors/](js/modules/errors/) - Error handling
  - [export/](js/modules/export/) - Export functionality
  - [eventListeners/](js/modules/eventListeners/) - Event listeners
  - [i18n/](js/modules/i18n/) - Internationalization
  - [userConfig/](js/modules/userConfig/) - User configuration

This modular design makes the project highly maintainable and extensible.
